สิ่งที่สนใจ

สายงานที่สนใจ อยากทำงานเกี่ยวกับ Data เป็นที่งฝั่ง Data science และ Data engineer โดยชอบทั้งการวิเคราะห์ข้อมูลทำ Machine Lerning และ Data Pipline


สนใจเกี่ยวกับการนำ BIG DATA มีประยุกต์ใช้กับ Marketing แล้วนำผลลัพย์ต่างๆ อาจจะผู้คนจำนวนมาก มาใช้ประยุกต์กับธุรกิจต่างๆ


สนใจเกี่ยววิทยาศาสตร์สมัยใหม่ การค้นพบต่างๆ เกี่ยวกับทางชีววิทยา


สนใจเกี่ยวเทคนิคการถ่ายทำภาพยนต์ ชอบการดูเบื้องหลังการถ่ายทำ เทคนิคต่างๆ ที่กองถ่ายทำ


ชอบการมองท้องฟ้าถ่ายรูปท้องฟ้า เพราะรู้สึกท้องฟ้าแต่ละวันสีไม่เหมือนกันเลยชอบถ่ายรูปเก็บเอาไว้เป็นความทรงจำ


ชอบเที่ยวดอย เดินป่า ทำให้รู้สึกสบายใจร่มรื่น ชอบดูชีวิตความเป็นของคนที่อยู้แถวนั้น
